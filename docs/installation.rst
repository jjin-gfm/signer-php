Installation
====================

Add the following information to your ``composer.json`` file (if you don't know
how ``composer.json`` works, `read the instructions
<https://getcomposer.org/doc/02-libraries.md#publishing-to-a-vcs>`__).

.. code:: json

    {
        "require": {
            "wepay/signer": "^1.0"
        }
    }
